# Memory Game

An interactive, browser-based memory game featuring customizable themes, flexible grid sizes, and local multiplayer support. Built with TypeScript, HTML5, and SCSS.

---

## 🚀 Features

- **Multiple Themes**:
  - **Code Theme**: Developer icons (HTML, CSS, JavaScript, TypeScript, React, Angular, Vue, Node, Python, Git, and more)
  - **Gaming Theme**: Retro & gaming motifs (Game Boy, Mario, Minecraft, Pac-Man, controllers, and more)
  - **DA Theme**: Custom design showcasing card motifs from my continuing education projects
- **Settings**:
  - Theme selection with dynamic preview
  - Pair count and grid size configuration (including a dynamic 4x4 or 6x6 grid for up to 36 cards)
- **Gameplay**:
  - Turn-based multiplayer for 2 players (Player Blue & Player Orange)
  - Score tracking (matched pairs per player)
  - Active player indicator with theme-specific icons
  - Card flip and selection animations (flip & wiggle effects)
  - Pause and leave game dialog
- **Endscreen (Results)**:
  - Dynamic winner determination and tie detection
  - Score evaluation via `sessionStorage`
  - Theme-specific artwork and confetti effects for the Code Theme

---

## 🛠️ Technologies

- **HTML5**: Semantic page layout
- **TypeScript (TS)**: Modular game logic, state management, and event handling
- **SCSS / CSS3**: BEM-structured stylesheets, CSS Grid & Flexbox, CSS transitions, and keyframe animations
